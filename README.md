# 📊 Sales & Revenue Analysis Dashboard

A comprehensive data analytics dashboard built with Python and Streamlit for analyzing sales and revenue data.

## 🎯 Project Overview

This dashboard provides interactive visualizations and insights for sales data analysis, including:
- KPI tracking (Revenue, Profit, Orders, AOV)
- Revenue trends and patterns
- Category and product performance analysis
- Regional sales distribution
- Detailed data exploration with filters

## 🚀 Features

### Data Import
- ✅ Upload CSV/Excel files
- ✅ Use built-in sample data
- ✅ Export filtered data

### Interactive Filters
- 📅 Date range selection
- 🏷️ Category filter
- 🗺️ Region filter
- 📦 Product filter

### Visualizations
- 📈 Revenue trend line chart
- 🥧 Category distribution pie chart
- 📊 Top products bar chart
- 🗺️ Regional performance chart
- 💰 Revenue vs Profit analysis
- 📦 Quantity analysis

### Analytics
- Key Performance Indicators (KPIs)
- Summary statistics
- Growth rate calculations
- Best performers identification
